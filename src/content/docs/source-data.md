    <h2>Catalogue Source</h2>
    <p>The catalogue is generated from <code>data/data.json</code> at build time. The JSON file remains the source of truth for message identifiers, messages, variations, descriptions, meanings, actors, contexts, communication methods, categories, and references.</p>
